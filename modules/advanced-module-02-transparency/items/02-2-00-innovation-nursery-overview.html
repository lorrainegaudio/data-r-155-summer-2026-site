Purpose

The 🪴 Innovation Nursery 🌳 Advanced activities help you build strong roots for future coursework by practicing two early habits that matter in R: creating reliable vector objects and setting healthy boundaries around AI use.

The Human-Centered AI activities ask you to explore when AI can support learning as a tutor, debugger, or translator, and when it starts doing too much of the thinking for you. Earned confidence in R comes from trying things yourself, keeping learning memos, asking questions, checking your work, and explaining what you did and why.

The Vector MadLib activities give you a playful way to grow those technical roots. You will create vectors, check their length and class, convert a character vector into a factor, and run your objects through MadLib story code. This theme is worth choosing if you want Advanced work that mixes reflection, creativity, and practical R habits you will use again later.

🪴 Innovation Nursery Learning Goals

By completing this theme, you will practice making thoughtful choices about AI use while strengthening the vector skills that support later R work.

  • Evaluate appropriate AI use by deciding whether AI supports learning or replaces learning in realistic R coding situations.
  • Apply a responsible help-seeking workflow by using the pattern attempt → document → ask → verify when deciding how to seek help from AI or people.
  • Build and inspect vector objects by creating numeric, integer, logical, and character vectors, then checking their length and class before using them in later code.
  • Use exact object names so later code can find and reuse the vectors you created.
  • Document reproducible learning by using an R Notebook to show code, results, short memos, verification steps, and learning struggles in one organized file.

Activities in This Theme

This theme has five activity pages. The Human-Centered AI activities work together as a guided reading plus a small-group discussion. The Vector MadLib activities work together as a coding challenge plus a short Canvas submission. You do not have to complete every Advanced option in the course, but each completed item in this theme can help you build toward your Advanced total.

  1. 🧘 Human-Centered AI Guided Activity: Appropriate AI Vignette

    Read and analyze AI-use scenarios.

    • Time estimate: 1 hour
    • You will produce: reading notes or decision rules for discussion
    • Why it matters: AI can help you learn R, but only if you stay responsible for trying, checking, and explaining the work.
  2. 🧘 Human-Centered AI Reflective Discussion: Initial Post

    Create your own vignette about AI use in an R learning situation.

    • Time estimate: 30 minutes
    • You will produce: one initial discussion post
    • Why it matters: writing a realistic vignette helps you recognize the line between responsible help-seeking and outsourcing the learning.
  3. 🧘 Human-Centered AI Reflective Discussion: Reply Post

    Evaluate classmates’ AI-use vignettes.

    • Time estimate: 30 minutes
    • You will produce: reply posts
    • Why it matters: evaluating another person’s scenario helps you practice giving a clear reason, condition, and verification plan.
  4. 🤪 Vector MadLib Guided Activity

    Build vectors and use them to generate randomized MadLib stories.

    • Time estimate: 1 hour
    • You will produce: a completed R Notebook with two generated MadLib stories
    • Why it matters: the story code depends on the objects you create, so this activity gives you practice with exact names, object checks, vector classes, and troubleshooting.
  5. 🤪 Vector MadLib Textbox Submission

    Submit your final story output and transparency memo.

    • Time estimate: less than 30 minutes
    • You will produce: MadLib result and short transparency memo
    • Why it matters: this submission shows that you ran the code successfully, checked your output, and can explain one issue you verified, fixed, or learned from.

Unlock Requirements

This theme unlocks after you earn ✅ Complete on the required 🌱 Core Module 1 assignments. When the theme unlocks, choose the activities that fit your goals, interests, and weekly schedule.

Reminder: Unlocking the theme gives you access to the activities. It does not remove the due dates. Advanced activities have deadlines and no token options, so plan your choices carefully.

Guiding Questions and Instructions

Use these questions to decide if and how you want to approach the Innovation Nursery activities.

Forethought / Before You Begin

  1. Which part of this theme interests you more right now: setting AI boundaries, building vectors, or both?
  2. What would make this Advanced work worth your time this week?

Performance / While You Work

  1. What evidence will show that you have achieved your learning goals?
  2. If you get stuck, what will you check first before asking AI or another person for help?

Reflection / After You Finish

  1. What did this theme help you understand about earned confidence in R?
  2. What did you learn about the boundary between helpful support and doing the learning work yourself?

Before You Leave This Overview

  • Start with 02.2.01 🧘 Human-Centered AI Guided Activity: Appropriate AI Vignette if you want to focus on AI boundaries and help-seeking.
  • Start with 02.2.04 🤪 Vector MadLib Guided Activity if you want to focus on vector practice and creative coding.
  • Use the discussion activities to test your thinking about AI use with classmates.
  • Use the MadLib activity to practice exact object names, vector checks, and transparent troubleshooting.
