Purpose

Review the supplemental resources on this page if you would like an alternative explanation, example, or walkthrough for concepts in this chapter, or if you need extra help with a specific skill.

These materials are optional. They do not replace the Chapter 16 guided activity.

Resources

  • Data Science: A First Introduction — 1.10 Exploring data with visualizations
    • This reading is most useful for learning the basic ggplot workflow for a bar chart: mapping variables with aes(), using geom_bar(stat = "identity"), then improving readability with xlab(), ylab(), and reorder() to sort categories.
    • It also helps clarify that tidy data structure is what makes ggplot work cleanly in the first place.
  • R for Social Scientists — 5 Data Visualization with ggplot2
    • This reading is most useful for understanding layers and geoms in practice: how the same ggplot(...) setup can become a scatterplot with geom_point() or a boxplot with geom_boxplot().
    • It also clearly shows the difference between mapping a variable inside aes(color = village) and setting a fixed color, plus when geom_jitter() is better than plain points because of overplotting.
  • Data Analysis and Visualization in R for Ecologists — Data visualization with ggplot2
    • This reading is most useful once you can already make a basic plot and need help with the workflow for refining it: layering global versus geom-specific aesthetics, using alpha for overplotting, combining geom_boxplot() with geom_jitter(), and remembering that the + belongs at the end of the line.
    • It also helps clarify when to move from one plot to many with faceting, such as facet_wrap() and facet_grid(), and how to adjust the appearance with a theme such as theme_bw().

Data Visualization: Short Video

Choose this video if you want a concrete example of the basic ggplot() workflow: mapping variables with aes() and then adding a graph layer such as geom_point(), geom_boxplot(), or geom_bar(stat = "summary", fun = "mean").

Watch for the coding step of piping a dataset into ggplot() and then switching to + to build the plot layer by layer, because that is a workflow move you will reuse in your own R work.

Data Visualization: Medium Length Video

Choose this video if you need a practical example of making bar charts for two categorical variables in ggplot, especially using geom_bar() with position = "dodge" or facet_wrap() to compare groups clearly.

Watch for the workflow of filtering the dataset first with filter() and drop_na(), then piping it into ggplot() so the plot only uses the categories you want.

Data Visualization: Detailed Video

Choose this video if you want a practical introduction to the basic ggplot() workflow: defining the dataset, mapping variables with aes(), and adding layers such as geom_point(), geom_line(), geom_boxplot(), or facet_wrap().

Watch for the coding step of deciding whether an aesthetic like color or size goes in the main ggplot() call or inside one geometry, because that controls whether the setting applies to the whole plot or just one layer.

How to Use These Materials

  • Start with the Chapter 16 guided activity before using these resources.
  • Use the first reading if you need help with the basic bar-chart workflow and readable labels.
  • Use the second reading if you are confused about geoms, layers, or mapping versus setting aesthetics.
  • Use the third reading if you need help refining plots with transparency, jittering, faceting, or themes.
  • Use the shortest video for a quick workflow refresher, and use the longer videos when you need more examples.
