Purpose

This Practice Space checks whether you can apply the core plotting skills from Chapter 16: ggplot2 after completing the ggplot2 Guided Activity.

In this chapter, you move from reading about plots to actually building them: creating plot objects, choosing the right geom, mapping variables with aes(), and explaining what a graph shows.

This light-weight practice matters because it is easy to communicate patterns with graphs in R. This is your chance to test your skills.

What This Practice Space Checks

  • Use ggplot() to build graphs from a dataset and aesthetic mappings.
  • Apply geoms such as geom_point(), geom_boxplot(), geom_histogram(), and geom_density() to match the goal of a graph.
  • Distinguish between different graph types, including when a density curve is more useful than a histogram.
  • Create and save plot objects with the required names: Pretty_Flowers, Petal_Spread, CO2_Graph, and Tall_Density.
  • Explain plotting choices and graph results in short written reflections using the course memo structure.

Task 4: Density Plot — Split + Transparency

🎯 Create a density plot using the built-in CO2 dataset and store the plot in an object called CO2_Graph. Put CO2 uptake on the horizontal axis. Use plant type to define separate smooth curves, so each plant type has its own density curve filled with a different color. Make the filled areas partially transparent so the curves can overlap and still be visible where they intersect. Finally, add the title “CO2 Uptake by Plant Type” using ggtitle() at the top of the plot.

# ⚡ Code
CO2_Graph <- CO2 %>%
  ________(________(x = uptake, fill = Type)) +
  ________(alpha = 0.5) +
  ________("CO2 Uptake by Plant Type")
# ✅ Checks
class(CO2_Graph)
CO2_Graph

🗣/🪞 What does the overlap between the density curves suggest about the two plant types?

Task 5: Histogram vs. Density

🎯 Create a graph from the heights dataset and save it in an object called Tall_Density. Put height, in inches, on the horizontal axis. Draw a histogram of heights using bars that are 1 inch wide, filled in blue with white borders. Scale the histogram so that the vertical axis shows density, not raw counts. On top of this, add a smooth density curve that shows the overall shape of the height distribution, using a dark orange line. Fill the area under that density curve with the same dark orange color, but make it partially transparent so you can still see the blue histogram bars underneath. Label the x-axis “Height (inches)” and the y-axis “Density”.

# ⚡ Code
Tall_Density <- heights %>%
  ggplot(________(x = height)) +
  ________(________(y = after_stat(density)), ________ = 1, ________ = "blue", ________ = "white") +
  ________(color = "darkorange", fill = "darkorange", ________ = 0.3, ________ = 1) +
  ________("Height (inches)") +
  ________("Density")
# ✅ Checks
class(Tall_Density)
Tall_Density
head(heights)
nrow(heights)

🗣/🪞 What does the density curve add to the histogram? Does the combined graph make the overall shape easier to describe?

Word bank: aes, geom_histogram, aes, binwidth, fill, color, geom_density, alpha, linewidth, xlab, ylab
