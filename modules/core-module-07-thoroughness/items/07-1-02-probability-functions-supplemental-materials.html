Purpose

Review the supplemental resources on this page if you would like an alternative explanation, example, or walkthrough for concepts in this chapter, or if you need extra help with a specific skill.

These materials are optional. They do not replace the Chapter 15 guided activity.

Resources

tidyr

  • Data Analysis and Visualization in R for Ecologists: Reshaping data with tidyr
    • This reading is most useful for clarifying when and why to switch between long and wide data, especially with pivot_wider() and pivot_longer(). It also helps connect a practical workflow: group_by() + summarise() first, then reshape the result so comparisons across groups are easier.
    • Look for how the same data changes structure when moving from long to wide and back again, and what information is carried in names_from, values_from, names_to, and values_to. Also notice where NA values come from after pivot_wider() and why filtering them after pivot_longer() can recover the original summarized data.
  • R for Social Scientists: Introducing dplyr and tidyr
    • This reading is most useful for clarifying how tidyr reshapes data to match the question you want to ask, especially with the older spread() and gather() workflow. It gives a concrete example of turning one categorical variable into multiple logical columns, then reversing that change when you need the variable back in a single column.
    • Look for how the data structure changes when values become column names versus when column names become values again. Also notice the practical pattern of using mutate() to create a helper logical column, then using filter() and select() to clean up the reshaped result.

How to Use These Materials

  • Start with the Chapter 15 guided activity before using these resources.
  • Use the normal distribution video if rnorm() or continuous simulated values feel unclear.
  • Use the Monte Carlo video if replicate() and repeated simulation feel abstract.
  • Use the tidyr readings only if the chapter or later work asks you to reshape data for comparison.
  • Return to your Quarto notes and connect each resource back to the simulation or probability question you are working on.
