🌱 Core | 30 min | Module 7

Purpose

This assignment collects your Chapter 16 ggplot2 Practice Space files so your work can be reviewed as a complete, reproducible record of what you built in the chapter.

Your uploaded files matter because they show your actual Quarto source, your saved workspace objects, and, if you render successfully, your formatted report for this ggplot2 practice.

File Upload

  • Required files to submit:
    • chapter16_practice.qmd
    • chapter16_practice.RData
  • Optional file to submit:
    • chapter16_practice.html
  • File types:
    • .qmd
    • .RData
    • optional .html
  • These files should show:
    • all 7 Practice Space tasks completed for Chapter 16
    • the actual code used for package setup, quick recall, scatterplot, boxplot, density plot, histogram with density overlay, and written reflection
    • the required memo structure for each task or code chunk
    • a workspace containing the required plot objects:
      • Pretty_Flowers
      • Petal_Spread
      • CO2_Graph
      • Tall_Density
  • Based on: Chapter-16_📈-ggplot2.html

How You Will Be Marked

✅ Complete if all are true

  1. Code + tasks
    • The uploaded files contain all the required Chapter 16 Practice Space work.
    • The submission is for the correct chapter and topic: ggplot2.
    • The files are not blank, cut off, or from a different assignment.
    • The .qmd file shows the actual code used for the chapter tasks.
    • The Practice Space work includes all 7 tasks:
      • package setup
      • quick recall
      • scatterplot
      • boxplot
      • density plot
      • histogram with density overlay
      • written reflection
  2. Memos
    • The file includes memos for every required task or code chunk.
    • Each task follows the course memo structure: 🎯/❓ Goal → ⚡ Code attempts → ✅ Checks → 🗣/🪞 Explain.
    • The file includes the short written explanations required in the quick recall and reflection tasks.
    • If outside help was used, that help is briefly disclosed in a memo.
  3. Reproducibility / chapter-specific requirements
    • The submission includes the required Quarto source file: chapter16_practice.qmd.
    • The submission includes the required workspace file: chapter16_practice.RData.
    • The workspace contains the four required plot objects: Pretty_Flowers, Petal_Spread, CO2_Graph, and Tall_Density.
    • The file includes package-loading code for ggplot2, dplyr, and dslabs.
    • If chapter16_practice.html is included, it opens as a complete rendered report.

🟡 Not Ready if any are true

  • No attachment, wrong file type, or file cannot be opened.
  • chapter16_practice.qmd is missing.
  • chapter16_practice.RData is missing.
  • Practice Space tasks are incomplete.
  • Code is missing from the Quarto file.
  • Required memos are missing.
  • The workspace does not contain the required ggplot objects.
  • The submission is for a different chapter or assignment.

If your submission is marked 🟡 Not Ready, revise to address the missing elements and resubmit for reevaluation before the assignment locks.

❌ Missing

No submission.

Before You Submit

  • Confirm your Quarto file is named chapter16_practice.qmd.
  • Confirm your workspace file is named chapter16_practice.RData.
  • Confirm both required files are uploaded.
  • Confirm your .qmd file includes all 7 Practice Space tasks.
  • Confirm your workspace contains Pretty_Flowers, Petal_Spread, CO2_Graph, and Tall_Density.
  • Confirm your memo notes explain what each graph shows, not just what code was used.
  • If you include chapter16_practice.html, confirm it opens correctly before submitting.
