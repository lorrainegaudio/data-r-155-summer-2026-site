Purpose

Missing values are one of the fastest ways for an analysis to become quietly wrong.

In this guided activity, you will practice built-in vector summaries, learn how to read help files for required arguments and defaults, detect and locate missing values, and make deliberate choices about what to do next with NAs.

That work is worth doing now because later analysis depends on checking assumptions instead of trusting default behavior without question. These skills will help you write code that is more accurate, more readable, and easier to defend in real data work.

Chapter

  • Chapter_6_❌MissingValues.html
  • Focus on the guided activity sections on special values, built-in functions, reading help files, the missing-values workflow, and nesting.
  • Pay close attention to how na.rm = TRUE changes summary results, how anyNA(), is.na(), and which() answer different questions about missingness, and how the chapter compares removing missing values with simple imputation.

Data

No external dataset required.

This guided activity works with small vectors such as Vector_NA and LargeVec so you can focus on missing-value behavior, summary functions, locating gaps, and comparing removal versus imputation without needing to import a separate data file.

Guiding Questions

  • How does the chapter distinguish between NA and NaN, and why does that difference matter for interpretation?
  • When a summary function returns NA, what does na.rm = TRUE change, and how can you confirm that from the help file?
  • How do anyNA(), is.na(), and which() help you move from “something is missing” to “I know where the problem is”?
  • What is the tradeoff between removing missing values and replacing them with a simple imputation like the mean?
  • As you follow the examples, how does nesting functions help you solve multi-step problems more compactly without losing track of what the code means?

You do not submit answers to these questions, but use them to focus your reading, coding, and memo notes as you work through the Guided Activity.
