Purpose

This assignment collects your Chapter 6 Missing Values Practice Space work as a rendered HTML report.

For this chapter, the uploaded file matters because it shows how you handled missing values in a reproducible R Notebook workflow: building a vector with NAs, checking what defaults do, locating missing values, using na.rm = TRUE, and comparing removal versus imputation with short memos that explain your reasoning.

File Upload

  • Required file to submit: chapter6_practice.html
  • File type: .html
  • This file should show:
    • your completed Chapter 6 Practice Space work for Task 1 through Task 8
    • the actual R code used in your R Notebook
    • your chapter-specific objects, including pool, scores_numbers, scores, n_missing, average_raw, average_scores, scores_is_missing, missing_positions, mean_of_five, today_date, now_time, scores_clean, and scores_imputed
    • your checks after code, such as length(), print(), sum(is.na(...)), mean(), and other chapter checks
    • your one-sentence written explanations where the chapter asks for them
    • your memo responses for each task
  • Based on: Chapter 6: Missing Values
  • If applicable, include: clear task headings, the required help-file response for ?sample, the explanation of why average_raw behaves the way it does using the word default, the one-sentence explanation of scores_clean using indexing and !, and the final reflection on a function object versus a function call.

How You Will Be Marked

  1. Code + tasks

    • The file is readable, complete, and not blank, cut off, or replaced with unrelated content.
    • The report shows the actual R code used for the chapter tasks.
    • The submission includes the chapter-specific missing-values workflow: building scores with 3 missing values, counting missingness, locating missing positions, computing a mean that still works when NA appears, removing missing values into scores_clean, and imputing missing values into scores_imputed.
    • The file includes the required chapter objects: pool, scores_numbers, scores, n_missing, average_raw, average_scores, scores_is_missing, missing_positions, mean_of_five, today_date, now_time, scores_clean, and scores_imputed.
  2. Memos

    • The file includes the required help-file explanation from Task 1 about how to tell required versus default arguments from the Usage line for sample().

    • The file includes the one-sentence explanation of what the code for scores_clean does, and that explanation mentions indexing and !.
    • The file includes the final reflection explaining the difference between a function object such as mean and a function call such as mean().
  3. Reproducibility / chapter-specific requirements
    • The submission is the correct output format: chapter6_practice.html, not the .Rmd source file.
    • The HTML report rendered successfully and opens as a complete report.
