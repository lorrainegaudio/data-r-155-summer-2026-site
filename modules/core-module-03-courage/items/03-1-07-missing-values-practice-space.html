Purpose

This Practice Space checks whether you can apply the main missing-values skills from Chapter 6: Missing Values after completing the associated Guided Activity.

In this chapter, you use help files to identify required and default arguments, build vectors with missing values, detect and locate NA values, use na.rm correctly, and compare removing missing values with simple imputation.

This practice matters because missing data can quietly break calculations, and you need to know how to check, explain, and handle those problems before moving on to more complex data analysis.

What This Practice Space Checks

  • Use a help file to distinguish required arguments from default arguments in a function such as sample().
  • Build a vector with observed values and missing values using seq(), sample(), and c().
  • Apply is.na(), which(), sum(), and mean() to detect, locate, and summarize missing values.
  • Use na.rm = TRUE correctly when a summary function would otherwise return NA by default.
  • Explain the difference between removing missing values, imputing missing values, and distinguishing a function object from a function call.
