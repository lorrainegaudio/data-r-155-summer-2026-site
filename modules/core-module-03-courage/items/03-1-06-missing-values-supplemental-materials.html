Purpose

Review the supplemental resources on this page if you would like an alternative explanation, example, or walkthrough for concepts in this chapter or need extra help with a specific skill.

These materials are optional and do not replace the chapter.

Resources

  • rmarkdown: Cheatsheet
    • This cheatsheet is most useful as a quick reference for the R Markdown workflow, especially how to create code chunks, use inline R code, and control output with chunk options like echo, eval, include, message, and warning. It is the page to keep open when you need the exact syntax for headings, lists, links, tables, images, or chunk options without searching through a longer guide.
  • Bookdown: 3.2 Notebook
    • This reading is most useful for clarifying the difference between an R Notebook and a regular knitted R Markdown document. Watch for the workflow difference: notebooks are previewed with the most recent chunk output, while ordinary R Markdown documents are knitted. Also notice that saving a notebook creates an .nb.html file alongside the source file.
  • R for Social Scientists: Functions and Their Arguments
    • This reading is most useful for learning how function arguments work, especially the difference between a required input and an optional argument with a default value. It is especially clear on how round() changes when you add digits = 2, and why naming optional arguments makes code easier to understand.

Video

Everyday Functions

This video is most useful if you need a practical set of everyday vector functions you can apply right away while exploring data in R.

It is especially helpful for seeing when to use mean(), max(), min(), length(), unique(), table(), and summary(), plus how na.rm = TRUE changes results when missing values are present. It also introduces functions like sample(), rnorm(), runif(), sort(), and order() for sampling and organizing values.

If this video does not appear, use the direct link: The Basics of R for Ecologists: Lesson 5 Part 1 (Other user functions for working with vectors) (11:37 min.).

Write Your Own Function

This video is most useful if you want a first look at writing your own functions in R instead of only using built-in ones.

It is especially helpful for seeing the workflow of defining a function with function(), setting an argument, using curly braces {}, returning a result with return(), and then applying that custom function to a vector.
