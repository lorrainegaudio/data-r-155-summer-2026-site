Purpose

This quiz checks your understanding of the Resilient Beginnings advanced module after you complete the guided activities connected to Chapter 1: RStudio Interface and Chapter 2: Scripts.

It focuses on early R workflow habits such as navigating RStudio, creating and naming objects, working in scripts, reviewing basic vector-building ideas, and recognizing how tenacity and careful review support real learning.

This practice matters because the course builds on these beginning skills, and small gaps here can make later R work much harder.

Static Quiz Preview

Important: This page helps you prepare for the quiz. You cannot submit quiz answers from this static site. When Canvas is available, complete the official quiz there.

What This Quiz Checks

  • Identify basic R syntax symbols and conventions used in beginning code, including assignment, comments, and common operators.
  • Distinguish valid object names from names that break R’s naming rules.
  • Recognize core parts of the RStudio interface and match them to their main purpose in a basic workflow.
  • Identify practical RStudio actions and shortcuts used to create scripts, run code, and check where files are being saved.
  • Apply foundational workflow knowledge of scripts, objects, and working directories to support accurate, reproducible work.

Review Quiz Steps

  • Time estimate: 30 minutes
  • Do: Complete after finishing the Resilient Beginnings guided activities in Module 1.
  • Before starting: Review Chapter 1 and Chapter 2.
  • Use: Your completed guided activities, your Chapter 1 and Chapter 2 course materials, and your own review notes.
  • Complete: All quiz questions, including the multiple-choice review questions and the fill-in-the-blank question.
  • Return to: This quiz until you reach a Complete score.

Which Pane

Create a Script

How do you create a new R Script in RStudio?

  • File > New File > R Script
