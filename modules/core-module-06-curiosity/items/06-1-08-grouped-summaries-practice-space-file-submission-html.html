Purpose

This assignment collects your Chapter 13 Grouped Summaries Practice Space work as a rendered HTML report.

For this chapter, the uploaded file matters because it shows how you used group_by(), summarize(), count(), and n() to produce grouped results, check your output, and explain what those grouped summaries mean in a reproducible Quarto workflow.

File Upload

  • Required file to submit: chapter13_practice.html
  • File type: .html
  • This file should show:
    • your completed Chapter 13 Practice Space work on grouped summaries and counts
    • the actual R code used for Task 1 through Task 6
    • your grouped-summary objects and outputs, including twinkle_twinkle, Olive_Garden, categorical_counts, and Infectious_Burden
    • your checks after code as needed
    • your inline memo sentences for counting unique diseases and states
    • your short reflection paragraph explaining why group_by() must come before summarize() when you want separate results for groups
  • Based on: Chapter 13: Grouped Summaries
  • If applicable, include: headings, section numbering, a table of contents, code that loads dplyr and dslabs, memoing with 🎯/⚡/✅/🗣 structure, and brief disclosure of outside help in a memo if you used it.
