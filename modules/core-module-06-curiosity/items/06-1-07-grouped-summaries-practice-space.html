Purpose

This Practice Space checks whether you can apply the main grouped-summary skills from Chapter 13: Group By, Summarize, and Count after completing the associated Guided Activity.

In this chapter, you use group_by(), summarize(), count(), and n() to describe data, compare groups, and explain what grouped results mean in words.

This practice matters because analysts often need to count categories, summarize numeric variables within groups, and produce reproducible work that others can follow and rerun later.

What This Practice Space Checks

  • Use group_by() and summarize() together to calculate separate results for each group instead of one result for the whole dataset.
  • Apply count() and n() to report frequencies and row counts for groups and category combinations.
  • Compute multiple summary statistics, such as mean and standard deviation, in a single summarize() call.
  • Distinguish between grouping variables and summary variables when analyzing categorical and numerical data.
  • Explain how grouping changes the meaning of a summary result and document your workflow in Quarto with checks, memos, and interpretation.

Task 4, Step 1: Make and Test a Prediction

How many rows would you expect if each state has 7 diseases?

If there are 51 states in the dataset, and you assume every state has data for all 7 diseases, how many groups will group_by(state, disease) create?

  • 51^7 groups
  • 51 - 7 = 44 groups
  • 51 / 7 ≈ 7 groups
  • 51 × 7 = 357 groups
  • 51 + 7 = 58 groups
