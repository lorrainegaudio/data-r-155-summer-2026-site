Purpose

This Practice Space checks whether you can apply the main logical filtering skills from Chapter 10: base R subsetting after completing the associated Guided Activity. In this chapter, you use relational operators, &, |, and %in% inside bracket subsetting to narrow the gapminder data to a short, defensible shortlist.

This practice matters because filtering rows and justifying your selection logic are common data-analysis tasks, and you need these skills before moving into more advanced data work later in the course.

What this Practice Space Checks

  • Use relational tests such as ==, <=, >=, and != inside bracket subsetting with df[rows, cols].
  • Apply & and | correctly to combine multiple row-filter conditions.
  • Use %in% to filter a data frame by membership in a set of allowed values.
  • Use negative indices to drop columns by position from a data frame.
  • Explain why parentheses matter when mixing & and | so your code matches your intended logic.
