Purpose

Review the supplemental resources on this page if you would like an alternative explanation, example, or walkthrough for concepts in this chapter, or if you need extra help with a specific skill.

These materials are optional and do not replace the chapter.

Resources

  • Advanced R: Logical Subsetting
    • This reading is most useful for understanding how logical vectors select rows when a condition is TRUE, which is the core idea behind filtering data in base R. Look closely at examples like x[x > 3] and df[df$x == 2, ], plus the warning that missing values in the index can produce NA in the output and that recycling logical vectors can cause mistakes if lengths do not match.

Video

Logical Operators

This video is most useful if you need a straightforward introduction to logical operators in R before using them to filter data or write conditional code. It is especially helpful for seeing how operators like <, <=, >, >=, ==, !=, !, &, |, and xor() produce TRUE or FALSE, and how those results connect to simple checks with variables and basic if statements.

If the video above does not appear, use this direct link: Introduction to LOGICAL OPERATORS in R (13:46 min).
