Purpose

Chapter 2 matters because it moves you from one-off Console commands to a script that records what you ran, in what order, and why.

In this chapter, you practice writing and running code from the Source pane, using comments to document your thinking, and building basic R objects such as vectors and a simple data frame.

This work is worth doing now because a script is the foundation of reproducible R work. You will need this habit before later chapters ask you to organize, check, and explain more complex analysis.

What This Practice Space Checks

  • Use seq() and : to build numeric vectors with the correct values.
  • Create character vectors with c() and extend an existing vector by adding a new element.
  • Check vectors with length() and mode() to confirm their structure.
  • Assemble a simple data frame with data.frame() from vectors you created earlier in the script.
  • Run and save an R script from top to bottom, while documenting errors and troubleshooting with short memos when needed.

Task 1

🎯 Build a numeric vector named exponents that contains the integers from 0.5 to 10 using the seq() function.

exponents <- ________(from = ________, to = ________)

length(________)
mode(exponents)

🗣 The output shows length 10 and mode numeric.

Word bank: seq, 0.5, 10, exponents

Task 2

🎯 Create a numeric vector using : called list_numbers that contains the integer values from 1 to 10.

________ <- ________ ________ ________

________(list_numbers)
length(________)

🗣 The length is 10 elements. The mode is numeric.

Word bank: list_numbers, 1, :, 10, mode, list_numbers

Task 3
