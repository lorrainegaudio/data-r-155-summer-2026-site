Purpose

This assignment collects proof that you installed and opened RStudio Desktop on your own computer.

For this course, that matters because all Core workflow is built around desktop RStudio, not Posit Cloud, not a browser-based workspace, and not the R application by itself.

This screenshot helps confirm that you are using the correct program before later assignments depend on it.

File Upload

  • Required file to submit: one screenshot image that shows RStudio Desktop open on your computer.
  • File type: .png, .jpg, or .jpeg.
  • This file should show:
    • the RStudio application window, not just a tiny cropped piece
    • the Console pane visible in the interface
    • enough of the RStudio interface that I can tell it is installed and running
    • the desktop program, not a browser tab or cloud workspace
  • Based on: Preface.html

Before You Submit

  • I read the Preface.
  • I installed R.
  • I installed RStudio Desktop.
  • I opened RStudio successfully.
  • My screenshot clearly shows the full RStudio window.

Important: Screenshots of Posit Cloud, another browser-based tool, or the R program alone do not meet this requirement.

How You Will Be Marked

✅ Complete if all are true

  1. Screenshot evidence
    • A screenshot file is uploaded successfully.
    • The screenshot clearly shows RStudio Desktop open and running.
    • The Console pane is visible.
    • The screenshot is large enough and clear enough to verify the correct program.
  2. Correct software
    • The screenshot shows the desktop version of RStudio used for this course.
    • The image does not show only the R program by itself.
    • The image does not show Posit Cloud or another browser-based workspace.

🟡 Not Ready if any are true

  • No screenshot submitted.
  • The screenshot does not clearly show RStudio running.
  • The screenshot is too cropped, too small, or too unclear to verify the program.
  • The screenshot shows the wrong program, such as R alone or a cloud/browser workspace.

If your submission is marked 🟡 Not Ready, fix the missing problem and resubmit before the assignment locks.

❌ Missing

No submission.

Helpful Reminder

This assignment is not asking for code yet. It is only checking that your RStudio Desktop setup is ready.
