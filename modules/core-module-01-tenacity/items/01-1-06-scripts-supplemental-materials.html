Purpose

Review the supplemental resources on this page if you would like an alternative explanation, example, or walkthrough for concepts in this chapter or need extra help with a specific skill.

These materials are optional and do not replace the chapter.

Resources

  • R for Social Scientists: Getting Started with R
    • This reading is most useful for practicing the early workflow of assigning objects with <-, writing comments with #, and checking vectors with length() and class(). It also gives concrete beginner examples of vector subsetting with square brackets and how NA affects calculations such as mean(rooms, na.rm = TRUE).
  • R for Social Scientists: Introduction to R — Subsetting Vectors
    • Use this reading to focus on subsetting vectors by position with square brackets, such as x[2], and on pulling multiple values by index. It is especially helpful for seeing how logical conditions inside brackets return only the values that match a rule.
  • Data Analysis and Visualization with R for Genomics: R Basics
    • This reading is most useful for clarifying the basic R workflow of creating objects with <-, running code from a script, and tracking results in the Environment pane. It also helps you see the difference between mode() and class(), along with basic arithmetic operations on objects.
  • Data Science: A First Introduction — 3.3 Data Frames, Vectors, and Lists
    • Choose this reading when you need a clear explanation of how vectors, lists, and data frames relate to each other in R. It is especially useful for understanding that a vector must contain one data type, a list can mix types, and a data frame is a special kind of list whose columns usually behave like vectors.

Video

This video is most useful if you want a concrete walkthrough of how objects and functions work together in R, especially assigning values with <- and then applying functions like sum(), hist(), summary(), class(), length(), head(), and tail() to those objects.

Choose it if you want to see how those ideas connect to a data frame workflow, including selecting one variable with $, making a quick plot or histogram, and handling missing values with an argument like na.rm = TRUE.

If the video above does not appear, use the direct link: R programming for beginners: using functions and objects in R (11:14 min.).
