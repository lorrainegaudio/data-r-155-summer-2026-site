🌱 Core | 1 hr | Module 1: 💪 Tenacity

Purpose

Learning R starts with learning how to move around RStudio, run commands, and understand where your work is going.

In this guided activity, you will identify the main panes in RStudio, use the Console for arithmetic and built-in functions, create named objects with <-, and manage your working directory and workspace.

That work is worth doing now because later analysis depends on knowing how to run code, store results, find your files, and avoid simple setup mistakes that can waste a lot of time. These skills form the foundation for later work with scripts, vectors, notebooks, and reproducible workflow in R.

Guided Activity

  • Time estimate: 1 hr
  • Do: Tuesday
  • Open: Chapter_1_📍Navigation.html
  • Use: RStudio, the Console, the Environment pane, the Files pane, and built-in R functions such as sum(), mean(), sd(), sqrt(), getwd(), save.image(), ls(), and load().
  • Have ready: RStudio installed on your computer and a dedicated course folder created on your computer so you can set it as your working directory.
  • Produce: A working R session with basic calculations, named objects in your Environment, and a saved workspace.RData file in your course folder.

Chapter

  • Chapter_1_📍Navigation.html
  • Focus on the Overview, the RStudio Interface section, the Console section on arithmetic and built-in functions, the Global Environment section on <- and object naming, and the Files pane section on working directories and saving or loading a workspace.
  • Pay close attention to the difference between seeing a result printed in the Console and storing a result as an object for later use.

Data

No external dataset required.

This guided activity uses simple numbers, built-in functions, and objects you create in the Console to practice navigation, object storage, naming rules, working-directory setup, and workspace management.

The main file you create during the chapter is a workspace.RData snapshot of your session.

Guiding Questions

  • How does the chapter distinguish the jobs of the Console, Environment, Source, and Files/Help panes in RStudio?
  • What is the difference between typing a calculation that prints a result in the Console and using <- to store that result as an object?
  • What object-naming rules matter most in this chapter, and which mistakes are likely to trigger beginner errors?
  • Why is the working directory so important when you save files, and what checks help you confirm that R is pointing to the correct course folder?
  • How does the chapter move from basic calculations and built-in functions to managing a saved workspace with save.image(), ls(), rm(), and load()?

You do not submit answers to these questions, but use them to focus your reading, coding, and memo notes as you work through the Guided Activity.

Before You Leave This Page

  • Confirm that you can identify the main RStudio panes.
  • Confirm that you can run arithmetic and functions in the Console.
  • Confirm that you can create named objects with <-.
  • Confirm that your working directory points to your course folder.
  • Confirm that you created and saved workspace.RData.
