Purpose

Review the supplemental resources on this page if you would like an alternative explanation, example, or walkthrough for concepts in this chapter or need extra help with a specific skill.

These materials are optional and do not replace the chapter.

Resources

  • RStudio IDE: Cheatsheet
    • Use this when you need help finding your way around RStudio. It is most useful for locating the Console, Source pane, Environment, Files, Plots, Packages, and Help tabs, and for learning the keyboard shortcuts you will use while writing and running code.
  • Data Carpentry for Social Scientists: Introduction to R
    • Focus on this reading when you need help with creating objects with <-, writing clear names, and building basic vectors with c(). It is especially useful for understanding the difference between an object name, a value, and a vector’s data type.
  • R for Social Scientists: Getting Started with R
    • This reading is most useful for practicing the early workflow of writing code in a script, assigning values with <-, using comments, and checking vectors with length() and class(). It also gives concrete beginner examples of subsetting with brackets and handling NA values with na.rm = TRUE.
  • R for Reproducible Scientific Analysis: Introduction to R and RStudio
    • Use this reading to clarify the basic RStudio layout and the difference between working in the Console versus saving code in an .R script. It is especially helpful for understanding why script-based work is better for rerunning code later.

Video

RStudio Workflow

This video is most useful if you want to see the early RStudio workflow in action: opening a script, running code with Run or Ctrl/Cmd + Enter, and using # comments so notes do not produce errors.

Choose it if you need a visual walkthrough of how to move between the script, Console, and Environment pane while doing simple calculations and assigning values with <-.
