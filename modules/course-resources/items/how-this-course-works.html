Start Here

This course is a 7-week online introduction to R programming. The course is organized around a steady weekly rhythm: learn by doing, practice in RStudio, submit evidence of your work, revise when needed, and choose Advanced activities when they fit your goals.

The most important thing to understand is this: 🌱 Core work comes first. Core work builds the required foundation for the course. 🌳 Advanced work gives you choices for deeper practice, extra challenge, and higher grade bundles.

The Big Picture

  • There are two learning tracks: 🌱 Core and 🌳 Advanced. Learn more here: How to Earn Your Grade: 🌱 vs 🌳 and ✅ vs 🟡.
  • The course runs on a repeated weekly rhythm. 🌳 Advanced activities unlock only after Core completion. Learn more here: 🗺️ Course Roadmap.
  • Core work has revision windows. If something earns 🟡 Not Ready, you are expected to revise. Learn more here: 🌱 Core Practice Space 🎟️ Token Bank.
  • Advanced work is for exploration and choice. It is not “the real course hidden somewhere else.” Start with Core, then choose Advanced activities that fit your goals and schedule.

What You Do Each Week

Most weeks include two 🌱 Core chapter sequences. Each sequence usually includes:

  1. A Guided Activity where you work through the chapter HTML in RStudio.
  2. A Practice Space Quiz where you check your understanding.
  3. A Practice Space File Upload where you submit your working file with code and memos.

After you make progress on Core work, you may choose 🌳 Advanced activities. Advanced work may involve discussions, peer review, screenshots, reflections, data work, Git/GitHub practice, AI-related activities, or longer file submissions.

How 🌱 Core Work Works

🌱 Core work is required for a C or higher. Core activities are designed to build fluency through repeated practice: type the code, run it, check the output, fix mistakes, and document what happened.

  • How to Complete 🌱 Core Guided Activities
  • How to Complete 🌱 Core Practice Space
  • 🌱 Core Practice Space 🎟️ Token Bank

Core rule: To earn a C or better, all 🌱 Core items must earn ✅ Complete.

How 🌳 Advanced Work Works

🌳 Advanced work is optional, but it is how you move from the C bundle toward the B or A bundle. These activities are usually more applied, more independent, or more reflective than Core work.

  • To earn a B: complete all 🌱 Core work and at least 12 🌳 Advanced assignments.
  • To earn an A: complete all 🌱 Core work and at least 18 🌳 Advanced assignments.

Advanced activities often build across parts, so start early and keep your files organized.

  • How to Complete 🌳 Advanced Guided Activities
  • How to Participate in 🌳 Discussion Groups

Advanced rule: Advanced work gives you choice, but it still has deadlines. There are no token options for Advanced work.

What ✅ Complete and 🟡 Not Ready Mean

This course uses Specifications Grading. Work is not graded by partial-credit point chasing. Instead, each item is checked against the requirements.

  • ✅ Complete means the work meets the specifications.
  • 🟡 Not Ready means revision is needed.
  • ❌ Missing means required Core work was not submitted.
  • ⭕ No Submission on Advanced work usually means you did not choose that option.

For the full grade-bundle explanation, review How to Earn Your Grade: 🌱 vs 🌳 and ✅ vs 🟡.

How to Get Help

Do not wait until the deadline if you are stuck. This course moves quickly, and small problems in R can turn into larger problems if you ignore them.

  • Re-read the Guided Activity or assignment prompt.
  • Check your spelling, punctuation, object names, parentheses, and file locations.
  • Ask peers in your discussion group when appropriate.
  • Message the instructor through Canvas Inbox when Canvas is available.
  • Book a meeting if you need a real-time conversation.

Bottom Line

Start with 🌱 Core. Complete the Guided Activity before Practice Space. Submit early enough to revise. Use Advanced work strategically. Keep your files organized. Ask for help before you are out of time.

This course rewards steady, careful work more than last-minute scrambling.
