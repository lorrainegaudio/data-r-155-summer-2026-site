Purpose

Review the supplemental resources on this page if you would like an alternative explanation, example, or walkthrough for concepts in this chapter or need extra help with a specific skill.

These materials are optional and do not replace the chapter.

Resources

Subsetting

  • R Documentation on Extract
    • This reference is most useful when you need the exact rules for [, [[, and $, especially the difference between selecting many elements versus one element. It is also the right place to check how negative, logical, and character indices work, and what drop = FALSE does when subsetting matrices or arrays.
  • Advanced R: Subsetting
    • This reading is most useful for building a stronger mental model of subsetting across vectors, lists, matrices, and data frames. It helps clarify when the same bracket syntax behaves differently across object types, which is exactly the point where many beginners start making indexing mistakes.

Code Chunk Options

  • knitr Chunk Options: Official List
    • Use this reference when you need the full list of chunk options and want to check whether an option exists or what it really does. It is especially useful for precise questions about options such as echo, eval, include, warning, message, and more specialized settings like results = 'asis'.
  • Posit R Markdown Cheatsheet: Code Chunks
    • This cheatsheet is most useful as a fast reminder of R Markdown code chunk syntax and the most common chunk options students actually use while writing reports. It is the page to keep open when you just need the syntax for a chunk header or a quick reminder of options like echo, eval, include, message, and warning.
  • R Markdown Cookbook: Chapter 11 Chunk Options
    • This reading is most useful when you are ready to go beyond the quick syntax and understand how chunk options change document behavior in a more detailed way. It is especially helpful for a deeper look at how to control whether code runs, whether code shows, and how results are printed in the final output.

Video

The Many Ways of Indexing

This video is most useful if you need a practical introduction to indexing and filtering data frames after working mostly with vectors.

It is especially helpful for seeing how to subset by row and column position, extract a column with $, filter rows with logical conditions, and use functions like head(), names(), rownames(), nrow(), ncol(), str(), levels(), grep(), and grepl() while exploring built-in data such as ChickWeight.

If the video above does not appear, use the direct link: The Basics of R for Ecologists: Lesson 6 Part 1 (indexing and filtering dataframes) (25:09 min.).
