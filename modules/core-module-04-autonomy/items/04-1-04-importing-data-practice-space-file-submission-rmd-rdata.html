Purpose

This assignment collects your Chapter 7 Importing Data Practice Space work as an R Notebook file and a workspace file.

For this chapter, those uploaded files matter because they show that you can manage file locations, load packages, bring data into R from Base R, a package, CSV, and Excel, and save your imported objects in a reproducible workflow that can be reopened and checked later.

File Upload

  • Required files to submit: chapter7_practice.Rmd, chapter7_practice.RData
  • File type: .Rmd, .RData
  • These files should show:
    • your completed Chapter 7 Practice Space work for the importing tasks
    • the actual R code used for Task 1 through Task 8
    • your file-management and setup work using getwd() and list.files()
    • code that loads readr, readxl, and dslabs
    • code that loads mtcars and movielens
    • code that imports Values_Transport_dplace.csv as dplace_df and car_prices.xlsx as car_prices_df
    • your saved workspace containing car_prices_df, dplace_df, movielens, and mtcars
  • Based on: Chapter 7: Importing Data
  • If applicable, include: the YAML header for chapter7_practice.Rmd, code chunks that run from top to bottom, and the required Task 1 memo about the difference between running file-path code in the Console versus in the knitted R Notebook.

How You Will Be Marked

✅ Complete if all are true

  1. Code + tasks
    • The uploaded chapter7_practice.Rmd contains the completed Task 1 through Task 8 work for Chapter 7.
    • The work is clearly for Chapter 7 importing data, not a different chapter or assignment.
    • The file is readable, complete, and not blank, cut off, or replaced with unrelated content.
    • The notebook shows the actual R code used for the chapter tasks.
    • The code includes the chapter-specific importing steps with getwd(), list.files(), library(), data(), read_csv(), and read_excel().
    • The submission includes the imported objects dplace_df and car_prices_df and the loaded datasets movielens and mtcars.
  2. Memos
    • The file includes the required Task 1 memo explaining the difference, if any, between file paths when you run the code in the Console versus when you Knit the R Notebook.
    • The memo is specific to your own workflow and not just copied from the chapter text.
    • Any short written notes used to explain checks, import issues, or fixes are included clearly in the notebook.
    • If outside help was used, that help is briefly disclosed in a memo.
  3. Reproducibility / chapter-specific requirements
    • The submission includes the correct files: chapter7_practice.Rmd and chapter7_practice.RData.
    • The notebook includes the Chapter 7 YAML header and is saved with the correct file name.
    • The code loads the required packages readr, readxl, and dslabs.
    • The notebook can be run from top to bottom without depending on missing objects from a previous session.
    • The .RData file contains car_prices_df, dplace_df, movielens, and mtcars.
    • The files document the chapter workflow clearly enough that the import process can be reopened, checked, and rerun later.

🟡 Not Ready if any are true

  • No attachment.
  • Wrong artifact.
  • The file cannot be opened.
  • Only one required file is submitted.
  • The .Rmd file is blank, incomplete, cut off, or unrelated to Chapter 7.
  • The .RData file does not contain the required imported or loaded objects.
  • The notebook does not show the required importing workflow.
  • The required Task 1 memo is missing or too generic to explain your own workflow.

If your submission is marked 🟡 Not Ready, revise to address the missing elements and resubmit for reevaluation before the assignment locks.

❌ Missing

No submission.

  • Confirm your workspace file is named chapter7_practice.RData.
  • Confirm both files are uploaded.
  • Confirm your notebook includes getwd() and list.files().
  • Confirm your notebook loads readr, readxl, and dslabs.
  • Confirm your notebook imports Values_Transport_dplace.csv as dplace_df.
  • Confirm your notebook imports car_prices.xlsx as car_prices_df.
  • Confirm your workspace contains car_prices_df, dplace_df, movielens, and mtcars.
  • Confirm your Task 1 memo explains your file-path or knitting workflow clearly.
