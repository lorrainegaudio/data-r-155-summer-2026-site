Purpose

This Practice Space checks whether you can apply the main importing skills from Chapter 7: Datasets and Importing Data after completing the associated Guided Activity.

In this chapter, you check file locations with getwd() and list.files(), load packages and built-in datasets, and import CSV and Excel files with read_csv() and read_excel().

This practice matters because most import problems are really file-management problems, and you need a reliable workflow for getting data into R before you can analyze it.

What This Practice Space Checks

  • Define tabular data as a table where rows represent observations and columns represent variables.
  • Interpret getwd() and list.files() to determine where R is looking for files and whether the needed files are visible.
  • Use data() to load datasets from base R and from an installed package.
  • Import a CSV file with read_csv() and an Excel file with read_excel().
  • Troubleshoot common import problems such as missing packages, wrong file locations, and incorrect file names.
