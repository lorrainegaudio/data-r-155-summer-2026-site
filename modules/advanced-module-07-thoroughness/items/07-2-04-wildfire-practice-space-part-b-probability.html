Purpose

This Practice Space Quiz checks whether you understand the main coding moves from Wildfire Guided Activity: Part B and the related chapter practice work. In this quiz, you will work with lognormal probability models for wildfire spread by using plnorm(), qlnorm(), dlnorm(), and rlnorm() to answer questions about tail probability, percentile distance, local likelihood, and simulated 8-hour spread distances in High-hazard areas. This practice matters because later simulation work depends on being able to interpret model assumptions, compute meaningful quantities from a distribution, and connect code output back to a wildfire question.

What this Practice Space Checks

This Practice Space checks whether you can:

  • Use plnorm() to compute and interpret a tail probability for High-hazard wildfire spread distance.
  • Use qlnorm() to identify a percentile-based run length under a lognormal model.
  • Use dlnorm() to compare the local likelihood of shorter versus longer wildfire runs.
  • Use rlnorm() and set.seed() to simulate 8-hour spread distances for ignitions in High-hazard areas.
  • Count, summarize, and visualize simulated fast-growing ignitions using logical conditions and a histogram.
