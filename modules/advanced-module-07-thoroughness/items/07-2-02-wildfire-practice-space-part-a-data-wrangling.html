Purpose

This Practice Space Quiz checks whether you understand the main coding moves from Part A of the Wildfire Advanced Guided Activity and the related chapter practice tasks. In this quiz, you will work with the early data-wrangling steps of the wildfire project: loading raw data, checking structure, creating quantile-based hazard categories, and summarizing statewide area shares. This practice matters because later modeling and simulation will only make sense if your data setup, category logic, and summary tables are correct.

What this Practice Space Checks

This Practice Space checks whether you can:

  • Use readr::read_csv() to load the wildfire sample data into the correct object.
  • Summarize a dataset to check rows, missing values, and area ranges.
  • Apply quantile() to create cutpoints from the whp variable.
  • Use mutate() and case_when() to turn a continuous hazard value into ordered categories.
  • Use group_by(), summarize(), and mutate() to compute area totals and category probabilities.

Task 4: Category Label

Analysts often turn a continuous variable into labeled groups so they can compare patterns more clearly across categories.

Add a Category label

🎯 Goal: Use dplyr::mutate() and dplyr::case_when() to turn the numeric whp values in county_raw into an ordered categorical variable called Category, using the quantile edges e you computed in Task 3. Store the resulting dataset as county_cats.

  2. Use mutate() to add a new variable called Category.

Task 5: Summary Table

After categories are created, you can summarize how much statewide area falls into each category and turn those area totals into probabilities.

Build prob_tbl from county_cats

🎯 Goal: Use dplyr::group_by() and dplyr::summarize() to aggregate total area by Category, then compute Probability as each category’s share of the total statewide area. Store the result as prob_tbl.

  1. Start from county_cats, which already has Category and cell_area_km2.
  2. Use group_by() so that all rows in the same Category are treated as one group.
  3. Use summarize() to compute area_km2 and n_cells for each category.
  4. Use .groups = "drop" so the result is no longer grouped after summarizing.
  5. Use mutate() to create Probability as area_km2 / sum(area_km2).
  6. Use select() to keep only Category, Probability, area_km2, and n_cells, in that order.
