Purpose

This Practice Space Quiz checks whether you understand the main coding moves from Wildfire Guided Activity: Part C and the related chapter practice work. In this quiz, you will work with a statewide wildfire simulation by building category, probability, and parameter vectors; simulating one season and many seasons; estimating the number of fast-growing fires; and interpreting the probability of seeing at least 50 fast-growing events in a season. This practice matters because it brings together probability, simulation, and interpretation in one complete workflow.

What this Practice Space Checks

This Practice Space checks whether you can:

  • Create ordered label, probability, and lognormal parameter vectors from a reference table.
  • Use sample.int() and indexed parameter vectors to simulate wildfire categories and distances for one season.
  • Use logical conditions, sum(), and mean() to count and summarize fast-growing fires.
  • Use braces and replicate() to repeat a one-season simulation across many seasons.
  • Interpret and visualize the distribution of seasonal fast-fire counts with a histogram and threshold line.

Task 5: Sample Categories for One Season

❓ In a season with 450 new ignitions statewide, what is the probability we see at least 50 fast-growing events, defined as fires spreading at least 10 km in 8 hours, when ignitions can occur in any hazard category according to their statewide area weights?

meanlog <- c(0.582, 1.046, 1.451, 1.907, 2.455, 2.866, 3.310)
sdlog   <- c(0.472, 0.325, 0.325, 0.280, 0.246, 0.220, 0.212)

N <- 450
S <- 100000
k <- 10

Task 8: Bundle One-Season Logic in a Single Block

Braces { ... } group several lines together so they act as one expression and return the final result.
