Purpose

This Practice Space checks whether you can apply the main vector skills from Chapter 3: Vectors after completing the associated Guided Activity.

In this chapter, you work in an R Notebook to build vectors, perform vector arithmetic, interpret recycling, check object class, work with character strings, and create logical vectors with comparison operators.

This practice matters because these are core mechanics you will keep using as the course moves from simple vectors to more complex data work, and you are also expected to document your thinking in a reproducible notebook workflow.

What This Practice Space Checks

  • Build numeric vectors with c() and check their contents and length.
  • Use vector arithmetic to combine vectors and interpret the result.
  • Recognize recycling when vectors of different lengths are used in arithmetic.
  • Explain how object metadata such as class() affects how R treats an object, including why mixed values in one vector may be coerced.
  • Create and interpret logical vectors using comparison operators such as >, <, ==, and !=.
