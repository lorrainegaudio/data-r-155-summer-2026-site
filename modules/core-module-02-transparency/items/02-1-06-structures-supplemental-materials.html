Purpose

Review the supplemental resources on this page if you would like an alternative explanation, example, or walkthrough for concepts in this chapter or need extra help with a specific skill.

These materials are optional and do not replace the chapter.

Resources

  • Bookdown: 3.2 Notebook
    • This reading is most useful for learning the practical R Notebook workflow: creating a notebook, running chunks interactively, and understanding that a notebook is previewed rather than fully knitted each time. It also helps clarify where chunk output appears and why saving creates an .nb.html file that keeps the most recent results alongside the source document.
  • Data Analysis and Visualization in R for Ecologists: 3 Exploring and Understanding Data
    • Use this reading when you need help understanding how R inspects data structures, especially with functions like str(), and how R treats factors as categorical data rather than plain text. It is especially useful for clarifying that factors may look like character vectors but behave differently because they store levels underneath.
  • Data Science: A First Introduction — 3.3 Data Frames, Vectors, and Lists
    • This reading is most useful for sorting out the difference between a vector, a list, and a data frame before you start more complex wrangling. It clearly shows that vectors must keep one data type, lists can mix types, and a data frame is a special kind of list whose elements must line up in length.

Data Types

This video is most useful if you need a clearer picture of data types in R and how to check or change them inside a data frame.

It is especially helpful for seeing the difference between character, factor, integer, numeric, and logical data, and for watching how functions like str(), as.factor(), as.integer(), and levels() are used to inspect and fix variable types.

If the video above does not appear, use the direct link: Data types in R programming (07:33 min.).

Modifying Vectors

This video is most useful if you need to see how named vectors, logical indexing, and modifying vector values work in practice.

It is especially helpful for watching how names() assigns labels, how logical tests inside square brackets return matching values, and how indexing can be used to replace selected values or set a value to NA without rewriting the whole vector.

If the video above does not appear, use the direct link: The Basics of R for Ecologists: Lesson 4 Part 3 (Naming and modifying Vectors + more indexing) (11:11 min.).
