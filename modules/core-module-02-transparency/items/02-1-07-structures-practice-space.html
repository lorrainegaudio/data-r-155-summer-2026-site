Purpose

This Practice Space checks whether you can apply the main vector-metadata skills from Chapter 4: Structures after completing the associated Guided Activity.

In this chapter, you practice using named indexing, positional indexing, and logical filtering on vectors. You also explain coercion and document your work transparently in an R Notebook.

This practice matters because it is the mechanics behind intentionally selecting values in R, and it prepares you for later filtering, recoding, and subsetting work with datasets.

What This Practice Space Checks

  • Use character indexing to retrieve values from a named vector by element name.
  • Apply positional indexing to retrieve specific elements from a vector by number.
  • Use logical filtering and indexing to create subsets of a vector based on a condition.
  • Explain why mixed-type vectors can trigger coercion, such as numeric values becoming character values.
  • Document troubleshooting, memo thinking, and verification steps in a reproducible R Notebook workflow.

Task 1

🎨 Indexing named vectors

🎯 Use character indexing to retrieve the hexadecimal color codes for "Matrix Green" and "Command Red" from the nerdy_palette vector.

# ⚡ Type this in your R Notebook
nerdy_palette <- c(
  "Octarine" = "#7F00FF",
  "TARDIS Blue" = "#003B6F",
  "Rebeccapurple" = "#663399",
  "Hooloovoo" = "#4F97A3",
  "Cosmic Latte" = "#FFF8E7",
  "Vantablack" = "#000000",
  "Matrix Green" = "#03A062",
  "Source Engine Magenta" = "#FF00FF",
  "Command Red" = "#CC0000",
  "Impossible Green" = "#4D9E68"
)

# ⚡ Type this in your R Notebook. Fill in the blanks.
nerdy_palette[c("________", "________")]

Answer format: fill in the two element names exactly as they appear in the named vector.

Task 3

⚠️ Logical filtering + indexing practice

First, create a vector called sales_q1 with four numbers of your choice. Create a vector called sales_q2 with four numbers of your choice. Then use vector arithmetic to create an object named growth that stores the difference between sales_q2 and sales_q1.

🎯 Using the growth vector, complete the following:

  1. Create a new vector positive_growth that contains only values in growth that are greater than zero.
  2. Create a new vector last_two that contains the last two elements of sales_q2. Use numeric positions, not names.
# ⚡ Type this in your R Notebook. Fill in the blanks.
positive_growth <- growth[growth ________ 0]

last_two <- sales_q2[c(________, ________)]

# ✅ Check the outputs
print(positive_growth)
print(last_two)

Answer format: fill in the comparison operator and the two numeric positions for the last two elements of a four-element vector.
