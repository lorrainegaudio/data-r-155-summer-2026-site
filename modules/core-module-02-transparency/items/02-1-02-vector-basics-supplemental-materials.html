Purpose

Review the supplemental resources on this page if you would like an alternative explanation, example, or walkthrough for concepts in this chapter or need extra help with a specific skill.

These materials are optional and do not replace the chapter.

Resources

  • R for Social Scientists: Introduction to R
    • Creating Objects in R — stop at functions and their arguments.
      • This reading is most useful for learning how to assign values with <-, choose clear object names, and understand that R is case sensitive. It also helps you see the difference between creating an object, printing it, and then using that object in later calculations.
    • Vectors and Data Types
      • Use this reading to focus on how to build vectors with c() and how R handles basic data types such as numeric, character, and logical values. It is especially useful for seeing what happens when values of different types are combined into the same vector.
  • R for Social Scientists: Getting Started with R
    • This reading is most useful for practicing the early workflow of writing code in a script, assigning objects with <-, using comments with #, and checking objects with functions like length() and class(). It also gives beginner-friendly examples of vector work, subsetting, and handling missing values with na.rm = TRUE.
  • R for Social Scientists: Introduction to R — Subsetting Vectors — stop at Missing Data.
    • This reading is most useful for learning how to pull values from a vector with square brackets, including selecting by position and selecting multiple elements at once. It also helps clarify how logical tests inside brackets can return only the values that match a condition.
  • Data Analysis and Visualization with R for Genomics: R Basics
    • Use this reading when you need a clearer picture of the basic R workflow: creating objects, applying functions, and inspecting what is stored in the Environment. It is especially useful for reinforcing how simple objects, vectors, and functions fit together in everyday R work.
  • Data Science: A First Introduction — 3.3 Data Frames, Vectors, and Lists
    • This reading is most useful for clarifying the difference between a vector, a list, and a data frame. It helps you understand that vectors usually hold one data type, lists can mix types, and data frames organize variables in columns for later analysis.
  • Data Analysis and Visualization with R for Genomics: Getting Help with R
    • Choose this reading when you need help learning the basic help workflow in R, especially using ?function_name, reading documentation, and checking a function’s arguments. It is most useful for building the habit of looking up what a function does before guessing at the syntax.

Video

Using Functions

This video is most useful if you need to see how objects and functions work together in R, especially creating objects with <- and then applying functions like sum(), summary(), class(), length(), head(), and tail().

It is also helpful for watching basic data frame workflow moves, such as using $ to work with one variable, making a quick plot() or hist(), and changing a function argument like na.rm = TRUE when missing values are present.

If the video above does not appear, use the direct link: R programming for beginners: using functions and objects in R (11:14 min.).

Fast Vectors

This video is most useful if you need to see faster ways to create vectors and then index specific elements with square brackets.

It is especially helpful for watching how :, rep(), and seq() build vectors, and how indexing can select values, exclude values with a minus sign, or create a cleaned version of a vector without overwriting the original object.
